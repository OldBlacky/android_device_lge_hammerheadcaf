## Build with Snapdragon llvm
Snapdragon llvm 3.8.8: [external link](https://github.com/SandPox/snapdragon_llvm)  
extract to prebuilts/clang/linux-x86/host/sdclang-3.8  
type "export SDCLANG=true" (without quotes)  
enjoy!
